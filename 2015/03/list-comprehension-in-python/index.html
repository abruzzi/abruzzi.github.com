区别

简单来说，两者都可以在迭代器上下文中使用，看起来几乎是一样的。不同的地方是generator可以节省内存空间，从而提高执行速度。generator更适合一次性的列表处理，比如只是需要一个中间列表作为转换。而列表推导则更适合要将列表保存下来，以备后续使用的场景。
